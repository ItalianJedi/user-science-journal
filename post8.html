Posted by Tony Graf on November 6, 2018

Evaluating Accessibility

The last two weeks I’ve used the website for Vinyl Me, Please to study and address usability issues. I want to continue to use Vinyl Me, Please for this week’s assignment which is to take a look at the accessibility of their website.

Analysis Tools

For my analysis I am going to use the WAVE Web Accessibility Evaluation Tool and the W3C Web Accessibility Initiative.

http://wave.webaim.org/

https://www.w3.org/WAI/test-evaluate/preliminary/

Results

Styles

Contrast

When filtering for WCAG 2.0 AA, we found:

  • 1 Error: Empty Link
  • 4 Alerts: 2 Skipped Heading Level, 1 Suspicious Link Text, 1 Noscript element (which is currently unchecked)
  • 28 Features: These mainly deal with alternative text
  • 42 Contrast Errors

Analysis and Recommendations

Vinyl Me, Please did very well when it comes to accessibility. Here are a couple of things they can do.

  1. The only real error this site had was a link at the top that contained no text. The user will not know the purpose of this link which is the logo of the website. Vinyl Me, Please should provide some text to this link to let the users know the functionality of that link.
  2. There were two alerts mentioning there were no heading. This can be addressed by adding a

    or

    tag so the user knows there is supposed to be a heading.
  3. There is one suspicious text link which just looks like it is out of context. It is recommended that the text link be reworded so there is more description of its destination.
  4. To address the 42 contrast errors, the contrast between the text and the background should increase.

Section 508

I did a quick comparison with the standards from Section 508. They are pretty much the same but W3C is a little more through. As we can see, the empty link error is unchecked for Section 508 while everything else remains the same. I will continue to use W3C for accessibility analysis.
