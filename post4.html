Reflections on Accessbility and Analytics

Posted by Tony Graf on November 26, 2018

Project: Gooooaaaal! (and Funnel)

This week we are tasked to set up goals using Google Analytics.

I thought the set up was fine it just took me a little bit of time to figure out what my goal was going to be. I used the tutorial from our class and from the Google Analytics tutorial on setting up goals and funnels.

My Goal

For this project I want to track how long people watch this video called What’s New In Web Accessibility. For roughly a half hour the presenter talks about automating web accessibility. I want to track that in my post and send it out to as many people as I can.

The presentation is about a Google project about automating web accessibility called Lighthouse. This project tests for SEO, best practices, and accessibility and has been integrated in Chrome Dev Tools. The presenter goes through an interesting walk through of his auditing of a website using Lighthouse. The video itself is 37 minutes but I want to see how long people watch the first section about Lighthouse which is about 20 minutes. I do not believe that many people will watch the full 20 minutes but I do think there is enough interesting content where a user will watch the first 5 minutes.

The second part is equally as import which is about the Accessibility Object Model, proposed as JavaScript API but for the purposes for this project I just want to focus on the first 5 minutes as pertaining to Lighthouse.

The goal I set up is called Lighthouse – Goal Duration Test and I set the details to track the goal of a duration of 5 minutes. Because it was a duration goal, I was unable to create a funnel since those are just for destination goals.

I also added an id to the link to track the events whenever someone clicked on the link as opposed to watched the video embedded on the page. I made the id of “lighthouse”.

I added the link to a couple of Slack channels though my school and am planning to send the link to friends, family and other ways to generate traffic.
